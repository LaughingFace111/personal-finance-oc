# 样式统一任务

## 目标
将"收入/支出页面"和"其他交易页面"的样式统一到"转账页面"的风格。

## 问题
之前的修改没有真正重构页面，只是加了几段英文说明文字。

## 转账页面风格特征（必须复用）
1. 页面主体是一个大白色圆角卡片容器
2. 卡片有白色背景、明显圆角、边框或轻阴影
3. 卡片顶部有清晰标题
4. 表单字段统一样式：宽度统一、高度统一、圆角统一、边框统一
5. 主按钮是明显的蓝色大按钮
6. 整体是卡片式、结构化、统一化的

## 必须完成的修改

### 1. 收入/支出页面
- 顶部导航区（汉堡按钮 + 标题）
- 大白色圆角卡片作为主体容器
- 卡片内：卡片标题 + 类型切换区 + 表单字段区 + 主按钮
- 删除 "Quick Entry" 等英文说明文字

### 2. 其他交易页面
- 同样结构：大白色圆角卡片
- 卡片内：卡片标题 + 分期/借出/借入切换区 + 表单字段区 + 主按钮
- 删除 "Special Entry" 等英文说明文字

## 执行步骤

### 第一步：分析
1. 收入/支出页为什么没改成功
2. 其他交易页为什么没改成功
3. 转账页的哪些结构应该复用

### 第二步：修改
- 先查看转账页的实际代码结构
- 抽公共表单组件/样式
- 重构收入/支出页
- 重构其他交易页

### 第三步：验证
- 三个页面风格是否统一

## 输出格式
1. 问题分析
2. 修改方案
3. 实际改动
4. 验证结果
5. 总结
